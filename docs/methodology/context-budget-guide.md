# Context Budget Guide（轻量版）

## 核心原则

- 不靠感觉管理上下文
- 不赌模型还能再装一点
- 先落盘，再继续

## 对长任务 agent 的最小规则

### 三个硬触发条件
1. **每完成一个独立交付单元** → 立即写 checkpoint
2. **session 结束前** → 必须写 checkpoint
3. **被中断恢复时** → 必须先读最新 checkpoint

### 默认裁剪规则
- `web_fetch`：默认 `maxChars <= 5000`
- `read`：单次默认最多 200 行
- 长命令输出：优先 `head -100` / `tail -100`

### 什么叫独立交付单元
- 一本书的拆书卡
- 一组 5 章精读
- 一章初稿
- 一份调研报告
- 一轮审稿报告

## 一句话准则

> 不允许连续跨两个工作单元只消费不产出。
